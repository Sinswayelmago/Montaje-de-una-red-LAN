Introducción proyecto

Presentación del proyecto


En este apartado veremos lo que necesitamos para poder empezar con nuestro proyecto de redes.
Vamos a seguir una estructura en la que podremos ver paso a paso lo que tenemos que hacer y
en el apartado de "Elaboracion proyecto" encontraremos como ejecutar estos pasos y los fallos
que hemos ido encontrando.

Montaje fisico


Primero, se harán las canalizaciones, montando las canaletas con sus respectivos enchufes.

Dispondremos de switches y un rack para enlazar y hacer funcionar esta red y crear una red local
ademas de poner una boca RJ45 cada 50 cm.

Etiquetar los racks y switches para no equivocarnos de donde irán nuestros cables para nuestra red.

Configuración de la red


Parametros de configuración


La red se configurara de manera manual, es decir en estático. Se cambiará el nombre a los equipos
como por ejemplo “windows-04-alba”.

Reiniciar y configurar las direcciónes IP en estático.

La puerta de enlace se dejará en blanco de momento y la dirección la dará el profesorado.

Parámetros

Dirección: 192.168.X.X
Máscara de subred: 255.255.0.0
DNS 1: 8.8.8.8
DNS 2: 8.8.4.4

Comprobación de red


Verificación de red

Utilizaremos dos comandos para comprobar que la red funciona correctamente.

Ipconfig: Es el que nos mostrará los parámetros de cada PC si está bien configurado.

Ping: Nos permite mandar paquetes de un PC host a un PC destino para comprobar
si la conexión se establece correctamente.
